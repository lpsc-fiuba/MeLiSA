# MeLiSA: Mercado Libre for Sentiment Analysis

*MeLiSA* is a dataset that contains +700M spanish and portuguese product reviews from xxxxxx users from 9 different countries of Latin America, taken from the [Mercado Libre site](http://mercadolibre.com/). Each of these samples contains a comment about a product written in portuguese or spanish, the title of that comment and the product rate (1-5).

Product reviews were obtained using the [Mercado Libre API](https://developers.mercadolibre.com/) and post-processed with the code provided in this repo.

For more documentation about the dataset checkout the [datasheet](./doc/datasheet.pdf) provided in [doc](./doc/) section.
